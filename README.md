6. LLM Integration

CIEL/Ω does not rely on LLMs for cognition.
Instead, it wraps them inside cognitive architecture.

Two backends are provided:

Primary Backend

Transforms internal kernel states and cognitive/affective metrics into natural-language output.

Auxiliary Backend

Evaluates the generated text for:

coherence,

helpfulness,

safety,

alignment with Λ₀ constraints.


The LLM becomes a linguistic expression module, not a reasoning engine.
